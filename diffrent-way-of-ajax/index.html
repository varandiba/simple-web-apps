Diffrent Way of AJAX

XMLHttpRequest: Old fashion way to send a GET/POST request to server.

Fetch: a Modern and easy to use way for send a GET/POST request to server.

jQuery: use jQuery library and API to send a GET/POST request to server.

Axios: a dedicated library for send a GET/POST request to server.

XMLHttpRequest: شیوه قدیمی برای ارسال درخواست های GET/POST به سرور

Fetch: شیوه‌ای مدرن‌تر و راحت تر برای ارسال درخواست های GET/POST به سرور

jQuery: استفاده از کتابخانه jQuery برای ارسال درخواست های GET/POST به سرور

Axios: استفاده از یک کتابخانه اختصاصی برای ارسال درخواست های GET/POST به سرور
